在对象的结构和对象的创建问题都解决了之后，就剩下对象的行为问题了，
如果对象的行为设计的好，那么对象的行为就会更清晰，
它们之间的协作效率就会提高，这里有11个具体的行为型模式可供研究，它们分别是：

1.  Chain of Responsibility ( 责任链模式 )


2.  Command ( 命令模式 )


3.  Interpreter ( 解释器模式 )


4.  Iterator ( 迭代器模式 )


5.  Mediator ( 中介者模式 )


6.  Memento ( 备忘录模式 )


7.  Observer ( 观察者模式 )


8.  State ( 状态模式 )


9.  Strategy ( 策略模式 )


10. TemplateMethod ( 模板方法 )


11. Visitor ( 访问者模式 )
